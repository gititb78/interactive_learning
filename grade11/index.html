מתכנתים, המשימה שלכם מתחילה עכשיו...

להלן פירוט מודולי הידע (Subroutines) שעלייך להטמיע במערכת המוח שלך כדי לעבור את השלב בהצלחה:

__סילבוס_משימה (Technical_Stack)

  • [שלב 01] הגדרת ישויות בסיסיות (עצמים פשוטים).
  • [שלב 02] ניהול דאטה ליניארי (מערך מספרים).
  • [שלב 03] ארכוב ישויות מורכב (מערך עצמים).
  • [שלב 04] ארכיטקטורת נתונים מתקדמת (עצמים מורכבים).

[ הנדסה_חברתית - המלצות המורה ]

  • Log File מסודר: וודאו שהמחברת שלכם מכילה את כל הקוד. אל תסמכו על ה-Cache של המוח.
  • קוד מקורי: יש להגיש שיעורי בית בזמן.
    • `CTRL+C` -> `CTRL+V` ממחברת של חבר *לא* נחשב קומפילציה.
    • (המערכת מזהה טביעת אצבע דיגיטלית זרה).
  • System Check: סומכת עליכם שתעשו עבודה נקייה.
