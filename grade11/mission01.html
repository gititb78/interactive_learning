> רקע מבצעי

סוכנים, הצלחנו ליירט תשדורת נתונים משרת עוין. התשדורת הגיעה אלינו בתור מערך חד-ממדי של מספרים שלמים. כדי לפרוץ למערכת שלהם, אנחנו חייבים לחלץ את "מפתח ההצפנה" מתוך רצף הנתונים.

> אלגוריתם הפיצוח

מחלקת המחקר שלנו גילתה שמפתח ההצפנה מסתתר בתוך "חריגות" (Spikes) ברצף.
חריגה מוגדרת כמספר שגדול ממש גם מהמספר שלפניו וגם מהמספר שאחריו במערך.
מפתח ההצפנה הסופי הוא הסכום של כל החריגות הללו.

// Example Transmission Data:
// int[] data = { 4, 8, 3, 5, 9, 2, 7 };

// Spikes identified:
// 8 (greater than 4 and 3)
// 9 (greater than 5 and 2)
// Encryption Key = 8 + 9 = 17

> דרישות המערכת (משימת התכנות)

כתבו פעולה (Method) שמקבלת את מערך הנתונים (מערך של שלמים בגודל N). הפעולה תסרוק את התשדורת, תאתר את כל החריגות, ותחזיר את מפתח ההצפנה המחושב.

אזהרת קומפילציה מגיטיט: שימו לב לגבולות המערך! חריגה מגבולות המערך (IndexOutOfRange) תפעיל את האזעקה של השרת ותכשיל את המשימה.
